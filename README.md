# Credit-Card-Default-L6-Data-Science-Professional-Summative
Public data science project exploring which customer and repayment behaviour characteristics appear the most linked with defaults using the dataset provided.
# Business Question
Which customer groups appear more likely to default, are there any patterns that can be identifies? 
# Dataset
This Project used a dataset from UCI Machine Learning Repository: Records - 30,000 , Columns/Features-23
# Tools Used 
Excel primarily
# Analysis 
The dataset was analysed where helper columns were needed before any pivot tables were made to support the analysis.
The Analysis shows findings of a strong correlation between lower limit bands = higher chance of PD, similarly but not so strongly, increase in age also shows signs of PD increasing.
Combined Segmentations added more context as when combining both segments this correlation proved through.
